Projects

Review Classifier

https://github.com/ashleyahn/Review-Classifier

Review Classifier is an implementation of a Naïve Bayes Classifier that categorizes movie reviews as positive or negative based off of the text in the review using simple machine learning.

Chikin Tinder

https://github.com/summer-hacks/Food-Picker

Chikin Tinder is a mobile application that was inspired by Tinder's "swipe left or right” UX. We use this Tinder-like experience to make the "where should we eat" decision-making process amongs a group of people easier and more fun, in which people get to "swipe" on restaurants based on their own preference and would receive matches after everyone finishes swiping. In this project, we utilized the Yelp API to populate restaurant data, including distance, pictures, and more.

Linx

https://github.com/ashleyahn/linx

Linx is a web application that organizes and save links that users can access from anywhere, inspired by online schooling due to COVID-19. Users can open selected links all at once and can organize and personalize their links by a filing system.
